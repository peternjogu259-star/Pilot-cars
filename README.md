**A Premium Automotive E-Commerce Experience**

Pilot Car Co. is a fully functional, 10-page web application designed for a luxury vehicle dealership serving the East African market. This project demonstrates advanced front-end development techniques, secure user authentication, and a dynamic shopping experience.



## üåü Key Features

### 1. Secure Authentication & Profile Management
* **Advanced Signup:** Implements a three-tier location selection system (Country > County > Constituency).
* **Security:** All user passwords are encrypted using `btoa()` encoding before being stored in `localStorage`.
* **Personalized Dashboard:** Displays user-specific data and delivery locations retrieved dynamically.

### 2. Dynamic Showroom & E-Commerce
* **Live Inventory:** High-definition car gallery rendered dynamically via JavaScript.
* **Persistent Cart:** A fully functional shopping cart that saves your progress even if the browser is refreshed.
* **Streamlined Checkout:** A multi-step process from cart review to shipping confirmation.

### 3. Order Tracking System
* Users can monitor the status of their high-value purchases through a dedicated "My Orders" tracking page.
